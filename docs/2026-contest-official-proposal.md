## 1. 서비스 개요 (필수, 매우 중요)

### 한 줄 정의
**이모추** (이번 주에 모하지 추천) — 한국관광공사 TourAPI와 Google Gemini AI가 위치·기분·동반자에 맞는 주말 나들이 코스를 10초 만에 설계해드리는 큐레이션 서비스.

### 왜 이모추인가
"주말에 뭐 하지?"는 매주 반복되는 고민입니다. 검색·블로그·인스타를 헤매다 결국 평소 가던 곳으로 돌아가는 일이 흔합니다. 이모추는 이 의사결정 피로를 단 5번의 탭으로 해결합니다. **위치·기분·시간·동반자·취향**을 알려주시면, AI가 한국관광공사의 실시간 공식 데이터를 분석해 동선까지 고려한 맞춤 코스를 그 자리에서 설계합니다.

### 핵심 차별점
1. **TourAPI 4.0 KorService2의 11개 엔드포인트를 통합 활용**
   관광지·축제·맛집·카페·문화시설·숙박을 모두 아우르는 코스 큐레이션. 단순 목록이 아니라 "오전엔 가벼운 산책, 점심엔 현지 맛집, 오후엔 카페" 같은 실제 시간 흐름이 담긴 코스로 응답.
2. **AI가 동선·시간·날씨를 동시에 고려**
   기상청 단기예보 API와 결합해 "비 오면 실내 위주", "맑으면 야외 명소" 처럼 날씨 반영. Gemini가 거리 계산·운영시간·편의시설(주차/유모차/반려동물)을 검토해 현실적인 일정 출력.
3. **공유 가능한 코스 URL — 카카오톡 한 번에**
   생성된 코스마다 고유한 영구 URL(`/course/{slug}`)이 발급되어 친구에게 카카오톡으로 공유. 받은 사람도 같은 화면을 보며 일정을 확인.
4. **Next.js App Router 고급 기능 채택 (Intercepting Routes)**
   장소 상세 페이지(`/spot/{id}`)는 모달 오버레이로도, 공유 가능한 풀 페이지로도 동시 작동. 인스타그램 사진 모달과 동일한 패턴으로, "공유 가능한 URL + 부드러운 모달 UX" 두 마리 토끼를 잡음.

### 운영 중 서비스 (상용 런칭 완료)
- **서비스 URL**: https://emochu.vercel.app *(실제 배포 도메인으로 교체 필요)*
- **기술 스택**: Next.js 16 App Router · React 19 · TypeScript 5 · Tailwind CSS v4 · Google Gemini · Supabase · Kakao Maps SDK
- **배포**: Vercel (자동 CI/CD)
- **사용 가능 기기**: 데스크톱 (1024px+ 매거진 split view) · 태블릿 · 모바일 (375px+ 단일 컬럼)

### 사용 시나리오 한 줄 요약
> 토요일 아침, "오늘 뭐 하지?" 검색 대신 이모추 → 5번 탭 → 10초 후 카카오톡으로 친구에게 코스 공유 완료.
